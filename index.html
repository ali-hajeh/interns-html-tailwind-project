Helpful Tips

  • Spacing: Use m-{size} for margin and p-{size} for padding
  • Colors: Use text-{color}-{shade} for text and bg-{color}-{shade} for backgrounds
  • Flexbox: Use flex, items-center, justify-center for alignment
  • Grid: Use grid grid-cols-{number} for grid layouts
  • Responsive: Use sm:, md:, lg: prefixes for responsive design
